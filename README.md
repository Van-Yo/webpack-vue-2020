# webpack-vue-2020
从零搭建webpack+vue脚手架

## 安装
- npm i
- npm run build
- 进入dist文件夹，新建index.html
- 在浏览器中打开dist文件夹中的index.html

index.html
```
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>webpack+vue+2020</title>
</head>
<body>
    <div id="app"></div>
    <script src="./bundle.js"></script>
</body>
</html>
```
